# Plan D — Classroom And Learning Journey

## Goal

Make the teacher and student flow feel like one coherent system:

- teacher creates
- teacher reviews
- teacher shares
- student joins
- student learns
- teacher sees progress

## Current problem

Pieces of this exist, but the journey does not yet feel tightly stitched:

- teacher home does not strongly foreground classroom work
- class detail is functional but not especially clear about next steps
- student entry is vulnerable to onboarding gates
- `classes` now redirects into library, which is technically correct but semantically muddy

## Teacher journey plan

### Home

Top actions should be:

- Create class
- Resume class generation
- Review class draft
- Share with students

### Class detail

The top of the page should answer:

- what stage is this class in?
- what is blocked?
- what should I do next?

Example next-step framing:

- `Curriculum not generated yet — generate it now`
- `Curriculum ready — review before sharing`
- `Class shared — 12 students joined`

### After creation

The moment after class creation should not just dump the teacher on a detail page. It should orient them:

- generated / in progress / waiting for approval
- what will happen automatically
- what action is available now

## Student journey plan

### Entry points

Student should be able to enter from:

- share link
- class code
- home continue-learning card

### Learning view

Once inside, the system should bias toward continuation:

- current lesson
- lesson progress
- quiz state
- next step

### Student home

Home should be about:

- Continue learning
- New assignments / joined classes
- Recent progress

Not about creation-first choices.

## Naming fix

If library is the canonical storage surface, keep it that way.

But when the user is doing classroom work, the UI should still speak their language:

- `My classes`
- `Continue class`
- `Join class`

Do not make the user translate everything into "library" to understand where they are.

## Why this matters

A classroom product wins when the teacher and student both know exactly what happens next.
